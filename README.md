# TPy
A python toolbox for transfer learning/domain adaptation methods

### Dependencies
- [scikit-learn](http://scikit-learn.org/)
- [scipy](https://www.scipy.org/)
- [numpy](http://www.numpy.org/)
- [cvxopt](http://cvxopt.org/)
- [cvxpy](http://www.cvxpy.org/)
